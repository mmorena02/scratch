Variables en Scratch

Clase 3: Aprendé a usar variables para guardar información, crear contadores, puntajes y vidas. El programa recuerda datos mientras corre.

📝 Datos del Estudiante

Caja de Herramientas

Las variables son cajitas con nombre que guardan un valor que puede cambiar. Se crean en la categoría Datos (naranja):

fijar [puntaje] a (0) cambiar [puntaje] por (1) mostrar variable [puntaje] esconder variable [puntaje] (puntaje) = (10) si <(puntaje) = (10)> entonces

💡 Regla clave: Siempre inicializá la variable con fijar a (0) al arrancar la bandera verde, antes de usarla.
